Usage: convert [options ...] file [ files... ] output_file<br>
<br>
Where options include:<br>
<ul>
<li>adjoin            =  join images into a single multi-image file</li>
<li>affine matrix     =  drawing transform matrix</li>
<li>antialias         =  remove pixel-aliasing</li>
<li>append            =  append an image sequence</li>
<li>average           =  average an image sequence</li>
<li>blur geometry     =  blur the image</li>
<li>border geometry   =  surround image with a border of color</li>
<li>bordercolor color =  border color</li>
<li>box color         =  color for annotation bounding box</li>
<li>cache threshold   =  megabytes of memory available to the pixel cache</li>
<li>channel type      =  Red, Green, Blue, Matte</li>
<li>charcoal radius   =  simulate a charcoal drawing</li>
<li>coalesce          =  merge a sequence of images</li>
<li>colorize value    =  colorize the image with the fill color</li>
<li>colors value      =  preferred number of colors in the image</li>
<li>colorspace type   =  alternate image colorspace</li>
<li>comment string    =  annotate image with comment</li>
<li>compress type     =  type of image compression</li>
<li>contrast          =  enhance or reduce the image contrast</li>
<li>crop geometry     =  preferred size and location of the cropped image</li>
<li>cycle amount      =  cycle the image colormap</li>
<li>delay value       =  display the next image after pausing</li>
<li>deconstruct       =  break down an image sequence into constituent parts</li>
<li>density geometry  =  vertical and horizontal density of the image</li>
<li>depth value       =  depth of the image</li>
<li>despeckle         =  reduce the speckles within an image</li>
<li>display server    =  obtain image or font from this X server</li>
<li>dispose method    =  GIF disposal method</li>
<li>dither            =  apply Floyd/Steinberg error diffusion to image</li>
<li>draw string       =  annotate the image with a graphic primitive</li>
<li>edge radius       =  apply a filter to detect edges in the image</li>
<li>emboss radius     =  emboss an image</li>
<li>enhance           =  apply a digital filter to enhance a noisy image</li>
<li>equalize          =  perform histogram equalization to an image</li>
<li>fill color        =  color to use when filling a graphic primitive</li>
<li>filter type       =  use this filter when resizing an image</li>
<li>flatten           =  flatten a sequence of images</li>
<li>flip              =  flip image in the vertical direction</li>
<li>flop              =  flop image in the horizontal direction</li>
<li>font name         =  font for rendering text</li>
<li>frame geometry    =  surround image with an ornamental border</li>
<li>fuzz distance     =  colors within this distance are considered equal</li>
<li>gamma value       =  level of gamma correction</li>
<li>geometry geometry =  perferred size or location of the image</li>
<li>gaussian geometry =  gaussian blur an image</li>
<li>gravity type      =  vertical and horizontal text placement</li>
<li>implode amount    =  implode image pixels about the center</li>
<li>intent type       =  Absolute, Perceptual, Relative, or Saturation</li>
<li>interlace type    =  None, Line, Plane, or Partition</li>
<li>label name        =  assign a label to an image</li>
<li>list type         =  Color, Delegate, Format, Font, Magic, or Module</li>
<li>loop iterations   =  add Netscape loop extension to your GIF animation</li>
<li>map filename      =  transform image colors to match this set of colors</li>
<li>matte             =  store matte channel if the image has one</li>
<li>median radius     =  apply a median filter to the image</li>
<li>modulate value    =  vary the brightness, saturation, and hue</li>
<li>monochrome        =  transform image to black and white</li>
<li>morph value       =  morph an image sequence</li>
<li>mosaic            =  create an mosaic from an image sequence</li>
<li>negate            =  replace every pixel with its complementary color</li>
<li>noise             =  add or reduce noise in an image</li>
<li>normalize         =  transform image to span the full range of colors</li>
<li>opaque color      =  change this color to the fill color</li>
<li>page geometry     =  size and location of an image canvas</li>
<li>paint radius      =  simulate an oil painting</li>
<li>ping              =  efficiently determine image characteristics</li>
<li>pointsize value   =  pointsize of Postscript font</li>
<li>preview type      =  image preview type</li>
<li>profile filename  =  add ICM or IPTC information profile to image</li>
<li>quality value     =  JPEG/MIFF/PNG compression level</li>
<li>raise value       =  lighten/darken image edges to create a 3-D effect</li>
<li>region geometry   =  apply options to a portion of the image</li>
<li>roll geometry     =  roll an image vertically or horizontally</li>
<li>rotate degrees    =  apply Paeth rotation to the image</li>
<li>sample geometry   =  scale image with pixel sampling</li>
<li>scale geometry    =  resize image</li>
<li>scene value       =  image scene number</li>
<li>segment values    =  segment an image</li>
<li>seed value        =  pseudo-random number generator seed value</li>
<li>shade degrees     =  shade the image using a distant light source</li>
<li>sharpen geometry  =  sharpen the image</li>
<li>shave geometry    =  shave pixels from the image edges</li>
<li>shear geometry    =  slide one edge of the image along the X or Y axis</li>
<li>size geometry     =  width and height of image</li>
<li>solarize threshold =  negate all pixels above the threshold level</li>
<li>spread amount     =  displace image pixels by a random amount</li>
<li>stroke color      =  color to use when stoking a graphic primitive</li>
<li>strokewidth value =  width of stroke</li>
<li>swirl degrees     =  swirl image pixels about the center</li>
<li>texture filename  =  name of texture to tile onto the image background</li>
<li>threshold value   =  threshold the image</li>
<li>tile filename     =  tile image when filling a graphic primitive</li>
<li>transparent color =  make this color transparent within the image</li>
<li>treedepth value   =  depth of the color color tree</li>
<li>type type         =  image type</li>
<li>units type        =  PixelsPerInch, PixelsPerCentimeter, or Undefined</li>
<li>verbose           =  print detailed information about the image</li>
<li>view              =  FlashPix viewing transforms</li>
<li>wave geometry     =  alter an image along a sine wave</li>
</ul>
<br>
By default, the image format of `file' is determined by its magic<br>
number.  To specify a particular image format, precede the filename<br>
with an image format name and a colon (i.e. ps:image) or specify the<br>
image type as the filename suffix (i.e. image.ps).  Specify 'file' as<br>
'-' for standard input or output.<br>
<br>
MMBase afkortingen:<br>
<ul>
<li>s           = geometry</li>
<li>r           = rotate</li>
<li>c           = colors</li>
<li>t           = transparent</li>
<li>i           = interlace</li>
<li>q           = quality</li>
<li>mono        = monochrome</li>
<li>highcontrast = contrast</li>
<li>flipx       = flop</li>
<li>flipy       = flip</li>
<li>dia         = negate</li>
</ul>
